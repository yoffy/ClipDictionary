# ClipDictionary

クリップボードに入っている英単語の辞書を引くユーティリティ。

## コンパイル環境

* Visual Studio 2017
    * C++/WinRT

## 実行環境

* Windows 10 (1803) 以降

## 対応している辞書

* [ejdic-hand](https://kujirahand.com/web-tools/EJDictFreeDL.php) (テキスト形式)
* [英辞郎 のテキストデータ](https://booth.pm/ja/items/777563)

## 使い方

1. 辞書を開く
2. 調べたい英単語をクリップボードにコピーする
